# 概要

iPhone の操作を自動化する「ショートカット機能」のレシピを投稿・検索できるサイトです（スマホ閲覧推奨）<br>
https://shortcutrecipe.com/

# 機能一蘭

-   ログイン
-   新規登録
-   パスワードリマインダー
-   レシピ登録・編集
-   ページネーション
-   レシピ検索
